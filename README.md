# dot-emacs.d
My emacs configuration.

## Introduction
The configuration should run on Emacs 26.3 or greater.

## Structure
```
~/.emacs.d
    | README.md
    | LICENSE
    | init.el
    | inits
        | init-global-settings.el   # Some basic global settings
        | init-elpa.el              # Initialize ELPA
        | init-xxx.el               # init-[package_name].el
```
